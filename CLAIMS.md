# Claims

Status: `scaffolded`

## Allowed Claims

| Claim | Status | Evidence |
| --- | --- | --- |
| This is a public GitHub scaffold for public-safe CAD and mechanical design studies. | `scaffolded` | Public GitHub repo, file tree, and validation notes. |
| Studies use synthetic or reviewed public-safe design content only. | `scaffolded` | Study files, export policy, and sealed-boundary notes. |
| Public routing requires proof-stack and human review. | `planned` | `VALIDATION.md` and `REVIEW_LOG.md`. |

## Claims Requiring Evidence

| Claim | Required evidence |
| --- | --- |
| A CAD export is public-safe. | Export review, redaction review, source classification, and human approval. |
| A design is manufacturable. | Manufacturing evidence, review, and explicit public approval. |
| A design is production-ready. | Formal engineering evidence and approval; not claimed here. |

## Forbidden Claims

Do not claim:

- production CAD release;
- production F3D availability;
- exact product dimensions;
- manufacturing package availability;
- product geometry release;
- customer part release;
- sealed geometry release;
- certification;
- deployment readiness;
- release, deployment, certification, or proof completion from public GitHub availability.
